# 🦇 The Batman — Projeto Web

![Batman Logo](https://petterpfs.github.io/The-Batman-Web-page-Project/src/logo_bat.png)

> Página web temática do filme **The Batman (2022)**, desenvolvida com HTML e CSS puro.

🔗 **[Acesse o projeto ao vivo](https://petterpfs.github.io/The-Batman-Web-page-Project/index.html)**

---

## 📋 Sobre o Projeto

Este projeto é uma landing page dedicada ao filme **The Batman (2022)**, apresentando a sinopse do filme, elenco principal, galeria de fotos, trailer e uma página de contato. O layout foi construído do zero com foco em design temático sombrio e imersivo, fiel ao universo do personagem.

---

## 🖥️ Páginas

| Página | Descrição |
|---|---|
| `index.html` | Página principal com sinopse, trailer e elenco |
| `pages/fotos.html` | Galeria de imagens do filme |
| `pages/contatos.html` | Formulário de contato |
| `pages/comentarios.html` | Seção de comentários |

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** — Estrutura semântica das páginas
- **CSS3** — Estilização e layout responsivo
- **GitHub Pages** — Hospedagem gratuita do projeto

## 🚀 Como Rodar Localmente

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/petterpfs/The-Batman-Web-page-Project.git
   ```

2. **Acesse a pasta do projeto:**
   ```bash
   cd The-Batman-Web-page-Project
   ```

3. **Abra o arquivo `index.html` no seu navegador** — não é necessária nenhuma instalação adicional!
